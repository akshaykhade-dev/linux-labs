# Linux System Administration Labs

This repository contains hands-on Linux administration labs based on real-world system administration tasks.
Each lab includes commands, explanations, and practical use cases.

## Topics Covered
- File & directory permissions
- User and group management
- Disk partitioning and LVM
- Mounting and persistent storage
- Backup and restore
- Cron jobs and scheduling
- Package and service management
- System monitoring and troubleshooting

## Operating Systems
- CentOS 6, 7, 8
- Ubuntu

## Goal
To demonstrate practical Linux system administration skills used in production environments.
